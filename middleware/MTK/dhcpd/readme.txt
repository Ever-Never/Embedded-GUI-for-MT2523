DHCPD module usage guide

Brief:          This module is an implementation of DHCP Server.
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/middleware/MTK/dhcpd/module.mk" in your GCC project Makefile.
                KEIL: Drag the middleware/MTK/dhcpd folder to your project. Add middleware/MTK/dhcpd/inc to INCLUDE_PATH.
                IAR: Drag the middleware/MTK/dhcpd folder to your project. Add middleware/MTK/dhcpd/inc to INCLUDE_PATH.
Dependency:     LWIP module must be enabled.
Notice:         N/A
Relative doc:   Please refer to internet and open source software guide under the doc folder for more detail.
Example project:Please find iot_sdk_demo project under project folder.
