ble_ancs_ios_notification
mt2523_hdk » apps » headset_ref_design
Overview
  • Feature description
    • This feature demonstrates a convenient approach to access notifications generated on iOS devices through Bluetooth LE link. The received notificaitons are shown in a pop up message.
Hardware and software environment
  • Please refer to mt2523_hdk/apps/headset_ref_design/readme.txt.
Directory contents
  • src/ble_ancs_ios_notification/ble_ancs_app.c: Notifications and attributes file.
  • src/ble_ancs_ios_notification/bt_ancs_common.c:Bluetooth stack callback file.
  • inc/ble_ancs_ios_notification/ble_ancs_app.h: Main header file.
  • inc/ble_ancs_ios_notification/bt_ancs_common.h:Common header file.
Run the feature
  • Build and Download
    • Please refer to mt2523_hdk/apps/headset_ref_design/readme.txt.
  • Power on the MT2533 EVB.
  • Download and install an app, such as "LightBlue", to connect to iOS devices that support Bluetooth LE.
  • Scan and find MT2533 on LightBlue, tap to start Bluetooth LE connection. On a pairing pop up window on iPhone, select "pair". A log, such as "[ANCSapp]New notification, type = 0, counter = 5\n" on MT2523 HDK indicates a successful operation.
