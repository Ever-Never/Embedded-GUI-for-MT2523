Introduction
Overview
  • This file guides you through the common and specific folder and file structure of available example projects. The example projects and applications are under [sdk_root]/project folder of the current SDK.
  • HAL template project is under the [sdk_root]/project/mtxxxx_hdk/templates without FreeRTOS. If you need to create a project with FreeRTOS, you can use [sdk_root]/project/mtxxxx_hdk/apps/freertos_thread_creation as a template.
Common folders and files
  • The common folders and files in example projects are listed below.
    • readme.txt: The first document to guide you through using this project, including setup, usage and structure of this project.
    • EWARM: IAR IDE related files, including the compiler dependent startup assembly sources and the project file.
    • GCC: ARM GCC compiler related files, including the compiler dependent files and project makefiles.
    • inc: The header files of this project including the following common header files.
      • cust_battery_*: The customization files of battery configuration (Not in hal_example projects).
      • cust_charging.h: The customization file of charging configuration (Not in hal_example projects).
      • ept*: The configuration files generated by Easy Pinmux Tool (EPT), with the *.dws file under ept_config folder of a project or under the mtxxxx_hdk/template/ept_config folder for the common EPT file.(May not in hal_example projects)
      • FreeRTOSConfig.h: FreeRTOS configuration. (Not in hal_example projects)
      • hal_feature_config.h: The configuration file of HAL features.
      • memory_map.h: The memory layout symbol file.
      • sys_init.h: The header file of sys_init.c initializing the system. (Not in hal_example projects)
    • MDK-ARM: Keil IDE related files, including the compiler dependent startup assembly sources and the project file.
    • src: The source files of this project.
Directory contents
  • Example projects using LinkIt 2523 HDK
    • mt2523_hdk/apps: Applications with FreeRTOS on MT2523x.
    • mt2523_hdk/hal_examples: HAL reference examples without FreeRTOS on MT2523x.
    • mt2523_hdk/templates: Templates for MT2523x.
    • mt2523_hdk/apps/iot_sdk_demo: A reference application with UI to demonstrate the basic functionalities of GNSS, Bluetooth, Sensor and watch.
  • Example projects using LinkIt 7687 HDK
    • mt7687_hdk/apps: Applications with FreeRTOS on MT76x7.
    • mt7687_hdk/hal_examples: HAL reference examples without FreeRTOS on MT76x7.
    • mt7687_hdk/templates: Templates for MT76x7.
    • mt7687_hdk/apps/iot_sdk_demo: A reference application to demonstrate the basic functionalities of Wi-Fi and internet protocols with command line interface (CLI).
