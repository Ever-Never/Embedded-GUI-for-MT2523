Overview
  • This is not a example project. This project is a bootloader for LinkIt 7687 HDK (with Firmware Over The Air update capability). The developer can refer this project to understand how to build a bootloader.
Hardware and software environment
  • Supported platform
    • LinkIt 7687 HDK.
  • PC/environment configuration
    • A serial tool is required, such as hyper terminal, for UART logging.
    • COM port settings. baudrate: 115200, data bits: 8, stop bit: 1, parity: none and flow control: off.
  • Customization Option
    • The BL_FOTA_DEBUG definition determines the bootloader accesses the UART driver directly to pring logs (overwrites printf(.)).
Directory contents
  • Source and header files
    • <sdk_root>/driver/board/mt76x7_hdk/bootloader/src/mt7687/main.c: Main program
    • inc/hal_feature_config.h: MT7687x's feature configuration file.
    • inc/flash_map.h: MT7687x's memory layout symbol file
    • GCC/syscalls.c: MT7687x's syscalls for GCC.
    • MDK-ARM/startup_mt7687.s: MT7687x's startup file for Keil.
    • EWARM/startup_mt7687.s: MT7687x's startup file for IAR.
    • src/system_mt7687.c: MT7687x's system clock configuration file
  • Project configuration files using GCC
    • GCC/feature.mk: Feature configuration.
    • GCC/Makefile: Makefile.
    • GCC/bootloader.ld: Linker script.
  • Project configuration files using Keil IDE
    • MDK-ARM/bootloader.uvprojx: uVision5 project File. Contains the project structure in XML format.
    • MDK-ARM/flash.sct: Scatter file.
  • Project configuration files using IAR IDE
    • EWARM/bootloader.ewd: IAR project options. Contains the settings for the debugger.
    • EWARM/bootloader.ewp: IAR project file. Contains the project structure in XML format.
    • EWARM/bootloader.eww: IAR workspace file. Contains project information.
    • EWARM/mtk7687_flash.icf: Linker script.
Run the example
  • Connect board to the PC with serial port cable.
  • Build bootloader with the command, "./build.sh mt7687_hdk bootloader" under the SDK root folder and download the binary file to LinkIt 7687 HDK.
  • Power on LinkIt 7687 HDK and bootloader log are showen on serial tool.
