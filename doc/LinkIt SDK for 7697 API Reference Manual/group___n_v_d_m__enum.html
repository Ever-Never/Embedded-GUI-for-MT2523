enum nvdm_status_t

This enum defines return type of NVDM APIs.

Enumerator
NVDM_STATUS_INVALID_PARAMETER 

The user parameter is invalid.

NVDM_STATUS_ITEM_NOT_FOUND 

The data item wasn't found by the NVDM.

NVDM_STATUS_INSUFFICIENT_SPACE 

No space is available in the flash.

NVDM_STATUS_INCORRECT_CHECKSUM 

The NVDM found a checksum error when reading the data item.

NVDM_STATUS_ERROR 

An unknown error occurred.

NVDM_STATUS_OK 

The operation was successful.
