Detailed Description

The MiniCLI command declaration structure.

The strcut cmd_s is defined to aggregate function pointer, help messsage and sub-commands for MiniCLI engine.

Note
Prefixing an ASCII value 01 to help message will effectively hide the command from user.
Warning
To reduce code size, duplicate commands are not detected by MiniCLI engine. It is developer's task to ensure there is no duplication in the command declaration.
